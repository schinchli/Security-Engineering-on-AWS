# AWS WAF CloudFront Pricing Guide

## Monthly Cost Breakdown

| Service | Usage | Cost |
|---------|-------|------|
| **AWS WAF Web ACL** | 1 Web ACL | $5.00 |
| **Managed Rule Set** | 1 rule group | $1.00 |
| **WAF Requests** | 1M requests | $0.60 |

**Total: ~$6.60/month**

## Cost Scenarios

| Configuration | Monthly | Annual |
|---------------|---------|--------|
| Basic (1 rule set) | $6.60 | $79 |
| Standard (3 rule sets) | $8.60 | $103 |
| Advanced (+ Bot Control) | $16.60 | $199 |

## Cleanup Cost

After cleanup: **$0** (WAF resources stop billing immediately)
